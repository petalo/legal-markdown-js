# Case Converter Helpers

These helpers convert text between common programming/naming conventions,
useful for generating identifiers, slugs, and formatted references in contracts.

Endpoint: <span class="legal-field highlight" data-field="api_endpoint">getUserProfileData</span>

## kebabCase

## pascalCase

Component: <span class="legal-field highlight" data-field="api_endpoint">GetUserProfileData</span>

## snakeCase
