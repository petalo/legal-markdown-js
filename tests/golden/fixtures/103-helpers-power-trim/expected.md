## Trim (alias for clean)

The `trim` helper removes leading/trailing whitespace, same as `clean`:

- Trimmed name: "<span class="legal-field highlight" data-field="padded_name">Maria Garcia</span>"
- Cleaned name: "<span class="legal-field highlight" data-field="padded_name">Maria Garcia</span>"
- Trimmed clause: "<span class="legal-field highlight" data-field="padded_clause">Confidentiality Agreement</span>"

Both `trim` and `clean` produce identical results for the same input.
